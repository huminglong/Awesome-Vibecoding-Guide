# Shadcn MCP服务器：用于AI辅助开发的专业UI组件

## 概述

Shadcn MCP服务器是一个强大的模型上下文协议（MCP）服务器，使AI助手能够通过自然语言命令浏览、搜索和安装专业组件库。它通过提供对shadcn/ui组件库的访问，弥合了AI生成的界面和生产质量设计系统之间的差距。

## 主要好处

### 1. 消除通用AI美学
- **问题**：AI生成的网站通常具有通用的、千篇一律的设计
- **解决方案**：访问具有一致样式的专业设计组件
- **影响**：将基本AI输出转换为生产就绪的界面

### 2. 专业质量的快速原型设计
- **好处**：在几分钟而不是几小时内创建美观、功能性的界面
- **优势**：组件使用可访问性、响应式和现代设计原则构建
- **结果**：可以演变为生产代码的专业外观原型

### 3. 一致的设计系统
- **功能**：所有组件遵循统一的设计语言
- **好处**：在整个应用程序中保持视觉一致性
- **价值**：减少设计决策并确保连贯的用户体验

## 安装

### 先决条件
- 安装Node.js 18+
- MCP兼容的AI助手（Claude等）
- shadcn/ui项目设置

### 安装步骤

1. **安装MCP服务器**：
```bash
npm install -g @shadcn/mcp-server
```

2. **配置您的AI助手**：
添加到您的MCP配置：
```json
{
  "mcpServers": {
    "shadcn": {
      "command": "npx",
      "args": ["@shadcn/mcp-server"],
      "env": {
        "SHADCN_PROJECT_PATH": "/path/to/your/shadcn/project"
      }
    }
  }
}
```

3. **在您的项目中初始化shadcn/ui**：
```bash
npx shadcn@latest init
```
