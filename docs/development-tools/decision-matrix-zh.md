# 工具选择决策矩阵

## 哲学提醒

本指南基于**成本效益原则**运作：
- **目标**：以免费或低成本获得80-99%的功能
- **真实来源**：本指南本身（在商业生产中经过验证）
- **方法**：从免费开始，仅在明确的ROI证明合理时升级

**昂贵工具陷阱：**
- 每月$200的Claude Code订阅对大多数自由职业者来说不是必需的
- 免费/预算工具可以交付专业结果
- 成本节省快速累积（每年节省$2,400 = 您不需要寻找的客户）

**当本指南与流行建议不同时：**
- 流行建议说"使用Cursor Pro"（$20/月）
- 本指南说"先尝试Qwen CLI（免费）"
- **原因**：大多数开发人员永远不需要高级功能

## 决策框架

在选择工具时使用这个四因素框架：

### 1. 成本效益
**问题**：交付价值的真实成本是多少？

**计算：**
```
成本效益 =（您实际使用的功能 × 可靠性）/ 总成本
```

**示例：**
- **Claude Code**：$200/月，100个功能，使用20个 → 20/200 = 0.1
- **Qwen CLI**：免费，40个功能，使用30个 → 30/0 = ∞

**赢家**：通常是更简单、更便宜的工具

### 2. 功能完整性
**问题**：它是否具有我实际需要的东西（而不是听起来很酷的东西）？

**基本功能检查清单：**
- ✅ 代码生成
- ✅ 上下文感知
- ✅ 文件编辑
- ✅ 错误理解
- ✅ 多文件重构（很好有）

**高级功能（很少需要）：**
- ❌ 语音编码
- ❌ 实时协作
- ❌ 高级分析
- ❌ 团队管理

### 3. 可靠性
**问题**：当我需要它时，它是否一致地工作？

**可靠性因素：**
- 正常运行时间（API可用性）
- 错误处理
- 速率限制
- 本地回退选项

**危险信号：**
- 频繁的"超过速率限制"错误
- 工作时间内的API停机
- 没有离线模式
- 模糊的错误消息

### 4. 集成
**问题**：它是否适合我现有的工作流程？

**集成检查清单：**
- 与我当前的IDE一起工作
- 支持我的技术栈
- 与我的git工作流程兼容
- 不需要工作流程更改
- 有逃生舱（未锁定）

### Qwen CLI（通过Factory AI免费）

**最适合：**
- 学习AI辅助编码的初学者
- 有预算意识的自由职业者
- 副项目和学习
- 想要CLI工作流程的开发者

**优点：**
- ✅ 完全免费（Factory AI提供免费层）
- ✅ 强大的模型（Qwen 2.5 72B）
- ✅ 大上下文窗口（128k令牌）
- ✅ 多文件编辑
- ✅ 简单CLI界面
- ✅ 不需要信用卡

**缺点：**
- ❌ 仅CLI（没有IDE集成）
- ❌ 速率限制（但对免费来说慷慨）
- ❌ 需要手动git命令
- ❌ 没有高级功能（代码审查、分析等）

**何时使用：**
- 开始AI编码
- 预算为$0
- 习惯CLI
- 不需要IDE集成

**设置：**
```bash
# 安装
npm install -g qwen-cli

# 配置
qwen config set apiKey YOUR_FACTORY_API_KEY

# 使用
qwen chat "创建联系表单组件"
```

## 迁移指南

### 从昂贵工具迁移到预算工具

#### Claude Code（$200/月）→ GLM（$15/月）

**节省：** $185/月（$2,220/年）

**您失去的：**
- 原生VS Code集成（改用Cline扩展）
- 一些高级上下文功能（95%的时间您不会注意到）

**您保留的：**
- 高质量代码生成（GLM-4.6优秀）
- 多文件编辑（通过Cline）
- 文件搜索和终端访问
- Git操作

**迁移步骤：**
1. **订阅GLM** - https://open.bigmodel.cn（约$15/月）
2. **安装Cline** - 在VS Code中，安装"Cline"扩展
3. **配置Cline** - 添加GLM API密钥
4. **测试工作流程** - 运行与Claude Code相同的任务
5. **取消Claude Code** - 一旦有信心

**调整期：** 1-2周适应新工作流程

**真实证言：**
> "我对从Claude Code降级持怀疑态度，但老实说我不想念它。GLM + Cline给我90%的功能，成本只有7.5%。" - 进行切换的自由职业者

#### Cursor Pro（$20/月）→ VS Code + 免费API

**节省：** $20/月（$240/年）

**您失去的：**
- 统一AI IDE（改用VS Code +扩展）
- Composer（使用Cline/Roo-Cline等效项）
- 无限高级模型（使用免费层或便宜API）

**您保留的：**
- 多文件编辑（Cline有这个）
- 代码生成（通过免费API）
- IDE功能（VS Code更强大）

**迁移步骤：**
1. **安装VS Code**（如果尚未使用）
2. **安装Cline或Roo-Cline**
3. **获取免费API密钥**：
   - Factory AI（免费层）
   - Anthropic（$5信用）
   - OpenAI（$5信用）
4. **使用API密钥配置扩展**
5. **一旦舒适就取消Cursor**

**专业提示：** 轮换免费层以最大化免费使用

#### Cursor BYOK → 保留Cursor，使用更便宜的API

**节省：** 可变（可能$10-15/月）

**策略：**
- 保留Cursor IDE（$20/月）
- 使用您自己的更便宜API密钥而不是Cursor的
- 仅支付API使用（可能$5-10/月）
- 净成本：~$25-30/月 vs $20/月 + 使用

**等等，那不是节省吗？**
- 实际上，Cursor Pro基于使用的定价可能超过$20/月
- BYOK给您成本控制
- 对简单任务使用更便宜的模型（GPT-3.5，Claude Haiku）
- 仅在需要时使用高级模型

## ROI场景

### 爱好者/学习者（目标：$0/月）

**概况：**
- 学习编码或构建副项目
- 没有客户收入
- 预算：$0

**推荐堆栈：**
```
IDE：VS Code（免费）或Zed（免费）
AI：Qwen CLI（通过Factory AI免费）
托管：Cloudflare Pages（免费）
分析：Plausible.io免费层
总计：$0/月
```

**能力：**
- 完整AI辅助开发
- 部署静态网站
- 基本分析
- 专业结果

**您放弃的：**
- 高级AI模型（使用免费层）
- 高级功能（反正很少需要）
- 无限使用（免费层足以学习）

**结论：** ✅ 完全可行，学习无妥协

### 自由职业者（预算：$5-20/月）

**概况：**
- 向1-5个客户收费
- $50-100/小时费率
- 收入：$2,000-5,000/月

**推荐堆栈（第1层：$15/月）：**
```
IDE：VS Code（免费）
AI：GLM（$15/月）
托管：Cloudflare Pages（免费）
分析：Plausible.io付费（$9/月）或免费
总计：$15-24/月
```

**ROI计算：**
```
月成本：$15
如果AI每月节省2小时
您的费率：$75/小时
节省：2 × $75 = $150
ROI：$150 - $15 = $135利润
年度：$135 × 12 = $1,620利润
```

**结论：** ✅ 10倍ROI，容易证明合理

**推荐堆栈（第2层：$20/月）：**
```
IDE：Cursor（$20/月）
AI：Cursor内置 + BYOK用于重度使用
托管：Cloudflare Pages（免费）
分析：Cloudflare Analytics（免费）
总计：$20-25/月
```

**何时选择第2层：**
- 您高度重视IDE集成
- 您想要多模型访问
- $20/月容易被您的收入证明合理

### 机构（预算：$50-100/月）

**概况：**
- 2-4个开发者的团队
- 向多个客户收费
- 收入：$20,000-50,000/月

**推荐堆栈（每个开发者）：**
```
IDE：Cursor（$20/月）或Claude Code（$200/月）
AI：包含在IDE中
托管：Cloudflare Pages（免费）或Pro（$20/月）
监控：付费层（$20-50/月）
每个开发者总计：$40-270/月
3人团队：$120-810/月
```

**选择哪个：**
- **Claude Code（$200/月）**如果：
  - 从事复杂项目
  - AI成本可向客户收费
  - 质量至关重要
  - 开发者收费$150+/小时

- **Cursor（$20/月）**如果：
  - 有成本意识的机构
  - 开发者收费$75-150/小时
  - 需要良好平衡

**ROI计算（Claude Code）：**
```
成本：每个开发者$200/月
如果AI每月节省10小时
开发者成本：$75/小时（完全加载）
节省：10 × $75 = $750
ROI：$750 - $200 = $550利润
每个开发者年度：$550 × 12 = $6,600
3人团队：$19,800/年利润
```

**结论：** ✅ 机构容易证明合理

### 何时昂贵工具有意义

**Claude Code（$200/月）在以下情况下合理：**

1. **您收费$150+/小时**并全职工作
   - 节省超过成本10倍+
   - 成本与收入相比可忽略不计

2. **从事企业/复杂项目**
   - 代码质量至关重要
   - 错误代价高昂
   - 客户期望最佳结果

3. **有可计费AI成本的机构**
   - 在项目报价中包含AI工具
   - 客户为工具付费
   - 对您的有效成本为零

4. **您需要特定功能**
   - 高级VS Code集成
   - 企业支持
   - 团队功能

**Claude Code不合理时：**

1. **您是爱好者** - 使用免费工具
2. **自由职业者收费<$100/小时** - GLM或Cursor更好的ROI
3. **不能直接报销** - 难以证明$2,400/年合理
4. **副项目** - 过度
5. **学习/教育** - 免费工具很好
