## 如何使用这些速查表

### 1. 快速参考
在开发期间在浏览器标签页中保持打开。

### 2. 复制粘贴
所有命令都可以复制粘贴。根据需要调整值。

### 3. 学习模式
理解模式，不要只是记忆命令。

### 4. 自定义
分叉这些速查表并添加您自己的项目特定命令。

### 5. 团队资源
与团队成员共享以获得一致的工作流程。

## 适合打印的版本

所有速查表都设计为打印良好：
1. 打开速查表
2. 打印（Cmd/Ctrl + P）
3. 保存为PDF或打印到纸张
4. 保持在桌面上以供快速参考
