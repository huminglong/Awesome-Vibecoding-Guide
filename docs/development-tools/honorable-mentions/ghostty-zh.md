# Ghostty — 现代终端模拟器 👻

## 概述
Ghostty是一个快速、功能丰富的终端模拟器，作为系统默认终端和Warp等更重选项的绝佳替代方案。以性能和简单性为设计理念，它为寻求更好终端体验而不需要不必要复杂性的开发人员提供了坚实的基础。

## 主要优势
- **轻量级和快速**：针对性能优化，资源使用最少
- **跨平台支持**：在macOS、Linux和Windows上工作
- **现代UX**：具有基本终端功能的清洁界面
- **系统终端替代品**：Terminal.app和类似默认设置的完整替代品
- **无AI依赖**：专注于核心性能的纯终端功能
- **积极开发**：定期更新和社区驱动的改进

## 最佳用例
- 替换系统默认终端（Terminal.app、命令提示符）
- 寻求Warp轻量级替代方案的开发人员
- 终端性能很重要的项目
- 喜欢简单工具而不是功能繁重选项的用户
- 跨平台开发工作流程

## 限制
- 没有内置AI功能（与Warp不同）
- 与既定终端相比集成较少
- 主题和插件的生态系统较小
- 可能缺乏成熟终端的一些高级功能
