# iTerm2 — 高级用户的终端

## 概述
iTerm2是macOS的终端模拟器，它做惊人的事情。它是Apple终端应用程序的替代品，但功能更强大。对于喜欢调整和自定义工具每个方面的开发人员，iTerm2提供了一个功能游乐场。它是现代终端（如Warp和Ghostty）的替代方案，特别是对于那些优先考虑对极简界面的细粒度控制的人。

## 主要优势
- **深度自定义**：终端的几乎每个方面都可以调整，从外观到光标的行为。
- **丰富的功能集**：分窗、热键窗口、搜索、自动完成等等。
- **AI集成**：内置AI聊天，可以与您的终端会话交互。
- **Shell集成**：与您的shell深度集成，以跟踪命令历史、当前目录等。
- **成熟和稳定**：悠久的开发历史意味着它是一个可靠的工具。

## 最佳用例
- 想要完全控制其终端环境的开发人员。
- 需要高级功能（如触发器、自动配置文件切换和内联图像）的用户。
- 对于那些发现Warp或Ghostty太受限或太固执的人来说，这是一个不错的选择。
- 自定义和脚本编写是关键的数据繁重工作流程。

## 限制
- **对新手来说压倒性**：大量的选项可能令人望而生畏。
- **过时的UI**：界面不如一些较新的终端那样现代或干净。
- **仅限macOS**：在其他平台上不可用。
