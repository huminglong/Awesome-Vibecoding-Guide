# 定价和经济学

## 基本定价策略

### 价格范围：$300-700+

**起始范围（您的前5个网站）**：$300-400
- 建立信心
- 创建作品集
- 完善流程
- 获得推荐

**标准范围（5-20个网站）**：$400-600
- 经过验证的记录
- 更快的交付
- 更好的质量
- 展示的作品集

**专业范围（20+个网站）**：$600-1000+
- 既定声誉
- 专业知识
- 高级定位
- 基于推荐的客户

## 按复杂性定价

### 简单单页网站：$300-500
**典型组件：**
- 英雄部分
- 服务概述
- 联系表单
- 地图/联系信息
- 隐私政策

**复杂性定义**：5-6个部分，单页或多页，最多主页 + 2个子页
**时间**：3-4小时
**示例客户**：杂工、清洁工、单一服务承包商

### 标准单页网站：$500-700
**典型组件：**
- 英雄部分
- 服务部分（详细）
- 定价表
- 作品集/画廊
- 推荐信
- 联系表单
- 地图/联系详细信息
- 隐私政策

**复杂性定义**：7-10个部分，通常最多5个子页
**时间**：4-6小时
**示例客户**：电工、水管工、暖通空调、景观美化

### 多页网站：$700-1200
**典型组件：**
- 主页（英雄、服务概述、CTA）
- 服务页面（详细细分）
- 作品集/画廊页面
- 关于页面
- 联系页面
- 隐私政策 + 服务条款

**时间**：6-10小时
**示例客户**：较大的承包商、想要内容分离的企业

### 高级/复杂网站：$1200+
**典型组件：**
- 多页中的所有内容
- 博客部分
- 自定义集成（预订、日历）
- 高级联系表单
- 多个服务类别
- 广泛的作品集

**时间**：10+小时
**示例客户**：既定企业、专业服务

## 地理定价考虑

定价因地点而异很大。以下是思考方式：

### 您的生活成本

**我的例子（欧洲国家）：**
- $300 = 租金 + 公用事业
- $1,000 = 4口之家的基本需求
- 目标：每月2个网站 = $1,000

**您的计算：**
1. 您的月生活成本是多少？
2. 您想要多少副业收入？
3. 您实际上每月可以构建多少个网站？
4. 价格 =（目标收入 ÷ 网站数量）+ 20%缓冲

### 市场条件

**高收入地区（美国、西欧、澳大利亚）：**
- 标准网站可以收费$800-1,500
- 客户有更大的预算
- 更多竞争，但更高的质量期望
